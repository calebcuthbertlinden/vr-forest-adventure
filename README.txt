SKY ADVENTURE
--------------------------------------------------------------------------------------
Welcome to Sky Adventure, where you can explore the wonderful floating paradise


------------------------------------------------------------------------
Controllers
------------------------------------------------------------------------
You will be required to use the Knuckles controllers for this VE

------------------------------------------------------------------------
Known bugs and work arounds
------------------------------------------------------------------------

- Falling over on start platform:
	-> Keep your hands close to you while the platform is lifting up

- Interactions pulling you in weird directions:
	-> AVOID TOUCHING THE JOYSTICKS FOR CHANGING DIRECTION
	-> If this already happened, use joystick to return to where you were facing in the beginning

- Getting stuck:
	 -> Restart and use the menu to navigate to where you need to be

- Struggling with interactions:	
	-> Release the trigger sooner
	-> Make sure you hand stays touching the rope
	-> Don't pull back too far
